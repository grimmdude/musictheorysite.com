What is Name That Chord?

Name That Chord is a free interactive online guitar chord finder and guitar chord identifier. It allows you to select frets and see which chord they make; this promotes experimentation and learning the notes on the fretboard. Use this tool alongside your guitar lessons and music theory lessons.

How It Works

Usage is pretty simple:

  • Click the notes on the fretboard to select/deselect them and see the results instantly.
  • Chords that match the selected notes exactly will show in the 'Exact Match Chords' line.
  • Chords that contain all the selected notes, but include other non-selected notes will show in the 'Partial Match Chords' line.
